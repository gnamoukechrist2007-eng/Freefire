# 🔥 Free Fire Community

Un réseau social complet pour la communauté Free Fire.

## Fonctionnalités

- 🔐 Connexion avec Google
- 📤 Publication de vidéos (liens YouTube)
- ❤️ Likes sur les vidéos
- 💬 Commentaires sur les vidéos
- 👤 Profil personnalisé
- 🔍 Recherche de vidéos YouTube
- 📱 Menu latéral (drawer)
